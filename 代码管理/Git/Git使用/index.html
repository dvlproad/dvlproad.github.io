  • Git-撤销（回退）已经add，commit或push的提交
  • SourceTree超前一个版本，落后N个版本

当你有多处备份的时候，你可以强制性的执行

git reset –hard HEAD^

但是你刚commit上去，未push出去的就没法找回了。
