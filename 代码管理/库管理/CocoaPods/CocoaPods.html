一、cocoapods 的安装和升级

1、查看pod版本

pod --version

2、安装 CocoaPods

sudo gem install -n /usr/local/bin cocoapods  # OS X 版本大于 10.11，使用此命令安装
sudo gem install cocoapods # OS X 版本小于等于 10.11，使用此命令安装

其他参考：百度地图中的CocoaPods文章

3、升级


sudo gem update cocoapods
// 以下该行命令可升级cocoapods
gem install cocoapods

// 移除现有pod
rm -rf /usr/local/bin/pod
// 重新下载pod
sudo gem install cocoapods

五、创建spec repo

  • 进入文件夹cd ~/.cocoapods/repos可以查看本地spec repo

  • 指定管理 lib repo的 specs repo的 url
    pod repo add dvlproad https://gitee.com/dvlproad/dvlproadSpecs.git
    注意 此处地址是spec仓库地址
